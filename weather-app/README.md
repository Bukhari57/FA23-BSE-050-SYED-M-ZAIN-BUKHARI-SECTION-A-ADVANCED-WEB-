# 🌦️ Node.js Weather Logger Application

 ** Node.js project** demonstrating:
- Node.js Architecture (Event Loop & Non-Blocking I/O)
- Core Modules (`fs`, `http`)
- NPM & `package.json`
- Third-party module usage (`axios`)

The app fetches live weather data from a public API, saves it to a file, and displays it via a local web server.
