# 🛒 Mini Online Store API

A simple and scalable RESTful API built with **Node.js** and **Express.js** following the **MVC (Model-View-Controller) architecture pattern**.

This project demonstrates:

- Express Routing
- Express Router (Modular Routes)
- Middleware (Global & Route-Level)
- Route Parameters
- Request Body Handling
- Proper Folder Structure (Scalable Architecture)
- 404 Error Handling

## 🏗 Architecture Pattern

This project follows the **MVC Pattern**:

- **Routes** → Handle incoming requests (like a receptionist)
- **Controllers** → Contain business logic
- **Middleware** → Process requests before they reach controllers
- **App.js** → Main server entry point

### 🍽 Restaurant Analogy

- **Client** → Customer placing order  
- **Middleware** → Waiter checking order & permissions  
- **Controller** → Chef preparing food  
- **Response** → Food delivered back to customer
